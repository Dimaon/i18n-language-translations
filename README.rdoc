= I18nLanguageTranslations

The purpose of this gem is to simply provide language translations.
The translations are extracted from Zend Framework data files.

This gem is based on [i18n-country-translations](I18nCountryTranslations)

== Installation

Add to your Gemfile:

    gem 'i18n-language-translations'

== Usage

  I18n.t(:US, :scope => :languages)
  
or
  
  t(:US, :scope => :languages)
